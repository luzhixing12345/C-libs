# 输出

- `\f` 换页,将活跃位置移至下一页开始处
- `\n` 换行,将活跃位置移至下一行开始处
- `\r` 回车,将活跃位置移至当前行开始处
- `\t` 水平制表, 将活跃位置移至下一个水平制表点

  相当于按Tab,补齐到8*n的位置

  ```c
  #include <stdio.h>
  int main() {
      for(int i=0;i<10;i++) {
          printf("123456 \t%d\n",i);
      }
      return 0;
  }
  ```

  ```bash
  123456  0
  123456  1
  123456  2
  ```

- `\v` 垂直制表, 将活跃位置移至下一个垂直制表点

  ```bash
  123
     456
  ```
